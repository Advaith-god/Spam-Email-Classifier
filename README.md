# Spam Email Classifier using Machine Learning

This project is a Machine Learning based Spam Email Classifier that identifies whether an email message is spam or not using Natural Language Processing (NLP).

## Features
- Text preprocessing using Bag of Words
- Spam classification using Naive Bayes algorithm
- User input prediction

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (CountVectorizer)

## How It Works
1. Email text is converted into numerical form
2. Model is trained using labeled data
3. New messages are classified as Spam or Not Spam
